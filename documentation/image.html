Image

Images are the heart of a website. Images convey message to the user faster than text.

Responsive Image

To use a image having full width and achieve responsiveness, use the predefined class responsive__img.

Round and Square Image

To use a round and square image use round__img or square__img. Note: Default size of round and square image is 150 pixel * 150 pixel.
