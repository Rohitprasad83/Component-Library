Welcome to Particle UI.

Particle UI is a component Library built using HTML , CSS and Javascipt. Particle UI helps you build web applications efficiently by using our pre-defined component classes.

Installation

To get started with Particle UI, Copy and Paste the below given stylesheet <link> into your <head> before all other stylesheets.

Note: For some component javascript is required, so copy the javascript snippet to the script tag of the HTML file or a separate js file.

Particle UI is built with the help of the following:

  1. Font Awesome - For icons
  2. Unsplash - For Images
  3. Coolors.co - For colors
